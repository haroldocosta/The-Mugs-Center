## Installation

```bash
$ npm install
```

## Running the database
```bash
$ npm start:dev:db
```

## Running the app

```bash
# development
$ npm run start:dev

# watch mode
$ npm run start:dev:nodemon

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
